# Telegram Bot - César Gómez

## Configuración

1. Instala las dependencias:
```bash
pip install -r requirements.txt
```

2. Obtén tu token del bot:
   - Habla con @BotFather en Telegram
   - Crea un nuevo bot con `/newbot`
   - Copia el token que te proporciona

3. Reemplaza `YOUR_BOT_TOKEN` en `bot.py` con tu token real

4. Ejecuta el bot:
```bash
python bot.py
```

## Funcionalidad

- Responde al comando `/start` con el mensaje de bienvenida
- Incluye un botón que enlaza a @CaesarGomezBot
